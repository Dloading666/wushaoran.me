# wushaoran.me

吴少然的个人作品集网站，面向 AI Agent 开发工程师实习岗位展示项目经历、技术栈和联系方式。

## Tech Stack

- React 19
- Vite 6
- Phosphor Icons
- CSS liquid glass UI

## Projects

- AI 简历助手：FastAPI + Agent Loop + Function Calling
- 合同审查助手：FastAPI + LangGraph + pgvector + Redis
- 临床诊疗问答助手：FastAPI + LangGraph + PostgreSQL + Neo4j

## Development

```bash
npm install
npm run dev
```

默认本地地址：

```text
http://127.0.0.1:5173/
```

## Build

```bash
npm run build
```
